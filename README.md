# Site Reinado React

Sistema web em React para organizar a memoria, os participantes, os locais e as edicoes do Reinado de Nossa Senhora do Rosario da Colonia Santa Isabel.

## Recursos

- Painel com indicadores e programacao de referencia.
- Cadastro de edicoes por ano.
- Cadastro de participantes.
- Cadastro de locais de celebracao.
- Organizacao de informacoes culturais.
- Consulta por busca e filtro de ano.
- Geracao automatica da proxima edicao.
- Acessibilidade com alto contraste e fonte ampliada.
- Dados salvos em `localStorage`.

## Rodar localmente

```powershell
npm install
npm run dev
```

## Build

```powershell
npm run build
```

Depois do build, a pasta `dist` pode ser publicada no GitHub Pages.
